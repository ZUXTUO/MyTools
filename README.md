# MyTools

A simple Android utility toolbox.

## Features

### 🔆 Brightness Control
Dim the screen below the system's minimum brightness level using a transparent overlay.

### 🔊 Volume Control
Reduce the audio volume below the system's minimum level for quieter playback.

### 🎲 True Random
Generate true random numbers using real-world entropy from device sensors (barometric pressure, gravity) and system states.

## Permissions
*   **Overlay**: Required for screen dimming.
*   **Audio Settings**: Required for volume reduction.
*   **Notifications**: Required for background service stability.
